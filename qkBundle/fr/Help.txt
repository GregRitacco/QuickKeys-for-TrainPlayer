*** Quick Keys ***

h est pour l'aide. Appuyez sur «h» pour ouvrir et fermer cette fenêtre.
u est pour les utilitaires. Appuyez sur «u» pour afficher le menu utilitaire.
f est pour plein écran. Appuyez sur «f» pour entrer ou sortir du mode plein écran.
z est pour pauZe. Appuyez sur «z» pour mettre l'action en pause. (Paramètres système)
j est pour Job Timer. Appuyez sur «j» pour démarrer et arrêter la minuterie.

*** Speed Switching ***

a est pour Auto-Follow. Appuyez sur «a» pour activer ou désactiver le suivi automatique.
t est pour Throttle. Appuyez sur «t» pour ajouter 1 mph (km / h) chaque fois que vous appuyez sur la touche.
g est pour Go. Appuyez une fois sur «g» pour la vitesse de chantier, à nouveau pour la vitesse maximale.
r est pour Reverser. Appuyez sur «r» pour changer la direction du train, même en mouvement.
s est pour Slow / Stop. Appuyez sur «s» pour ralentir à la vitesse de couplage, encore une fois pour arrêter.

Appuyez sur «u» pour sélectionner Phake Physics.
"
,
"
*** Phake Physics ***

a est pour Auto-Follow. Appuyez sur «a» pour activer ou désactiver le suivi automatique.
t est pour Throttle. Le taux d'accélération est augmenté chaque fois que vous appuyez sur «t».
b est pour le frein. La pression de freinage augmente chaque fois que vous appuyez sur la touche «b».
n est pour Neutre. Appuyez sur «n» pour maintenir la vitesse actuelle.
r est pour Reverser. À l'arrêt, «r» change la direction du train.
e est pour l'urgence. Appuyez sur «e» pour arrêter tout de suite.

Appuyez sur «u» pour sélectionner Changement de vitesse.
"
,
"
*** Drop Kick ***

c est pour Cut. Appuyez sur «c» pour couper la voiture sélectionnée. (Paramètres système)
k est pour Kick. Appuyez sur «k» pour donner un coup de pied à certaines voitures. (Frappez-les)
d est pour Drop. Appuyez sur «d» pour déposer des voitures. (Laissez-les partir)

Appuyez sur «k» ou «d» avec le train arrêté pour nettoyer les moteurs de coup de pied.
