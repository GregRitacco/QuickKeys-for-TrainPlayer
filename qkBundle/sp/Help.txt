*** Speed Switching ***

a es para Seguimiento automático. Presione 'a' para activar o desactivar el seguimiento automático.
Es para Throttle. Presione 't' para agregar 1 mph (kph) cada vez que presione la tecla.
g es para Go. Presione 'g' una vez para la velocidad de la yarda, nuevamente para la velocidad máxima.
r es para inversor. Presione 'r' para cambiar la dirección del tren, incluso mientras está en movimiento.
s es para Slow / Stop. Presione 's' para reducir la velocidad de acoplamiento, nuevamente para detener.

Presione 'u' para seleccionar Phake Physics.
"
,
"
*** Phake Physics ***

a es para Seguimiento automático. Presione 'a' para activar o desactivar el seguimiento automático.
Es para Throttle. La tasa de aceleración aumenta cada vez que se presiona 't'.
b es para freno. La presión de frenado aumenta cada vez que se presiona la tecla 'b'.
n es para Neutral. Presione 'n' para mantener la velocidad actual.
r es para inversor. Cuando se detiene, 'r' cambia la dirección del tren.
e es para Emergencia. Presione 'e' para detenerse ahora mismo.

Presione 'u' para seleccionar Cambio de velocidad.
"
,
"
*** Drop Kick ***

c es para Cortar. Presione 'c' para cortar el automóvil seleccionado. (Configuración del sistema)
k es para Kick. Presiona 'k' para patear algunos autos. (Patéalos)
d es para Drop. Presione 'd' para dejar caer algunos autos. (Déjalos ir)

Presione 'k' o 'd' con el tren detenido para limpiar los motores.
